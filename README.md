Email Spam Detection Using ANN

This project detects spam emails using an Artificial Neural Network (ANN).
It uses features like the number of words, links, attachments, sender reputation, and keywords. The model learns to classify emails as spam or not spam, helping to filter unwanted emails automatically.

Features
Email text analysis (words, links, exclamations, attachments)
Sender reputation scoring
Numerical and categorical features combined
ANN (Feedforward Neural Network) for classification
How to Use
Load the dataset.
Clean the data and handle missing values.
Split into training and testing sets.
Scale features and train the ANN.
Evaluate using accuracy, classification report, and confusion matrix.
Requirements
Python 3.x
pandas, numpy, scikit-learn
tensorflow / keras
matplotlib, seaborn
