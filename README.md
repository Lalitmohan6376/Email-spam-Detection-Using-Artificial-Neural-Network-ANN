# Email Spam Detection Using ANN



This project detects **spam emails** using an **Artificial Neural Network (ANN)**.  



It uses features such as the number of words, links, attachments, sender reputation, and keywords. The ANN model learns to classify emails as **spam** or **not spam**, helping automatically filter unwanted emails.  



## Features



- **Email text analysis** (words, links, exclamations, attachments)  
- **Sender reputation scoring**  
- **Numerical and categorical features combined**  
- **ANN (Feedforward Neural Network)** for classification  
- **Deep Learning model trained on a balanced dataset**  



## How to Use



1. Load the dataset  
2. Clean the data and handle missing values  
3. Drop unnecessary columns and balance the dataset  
4. Split the data into training and testing sets  
5. Scale numerical features  
6. Train the ANN model  
7. Evaluate using **accuracy**, **classification report**, and **confusion matrix**
   

## Requirements / Tools / Frameworks


- **Python 3.x**  
- **Libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn  
- **Deep Learning Framework:** tensorflow / keras
